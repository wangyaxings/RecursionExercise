总结：
（1）NxM中
正方形个数：
长度1的有N*M个
长度2的有（N-1）*（M-1）
长度3的有（N-2）*（M-2）
...
长度min(N,M)的有（N-min(N,M)+1）*（M-min(N,M)+1）
共有SumS = N*M+(N-1)*(M-1)+...（N-min(N,M)+1）*（M-min(N,M)+1）
这里用循环求解要比递归更好理解
（2）所有长方形（包括正方形）N为长，M为宽
宽为1的长方形有m种，宽为2的有m-1种...宽为m的有1种
长为1的长方形有n种，长为2的有n-1中...长为n的有1种
所以所有长方形共有m*(n + n-1 + ...+1) + (m-1)*(n + n-1 + ...+1)+...1*(n + n-1 + ...+1)
即总共有SumA=（m + m-1 +... 1）*(n + n-1 + ...+1)种
（3）除了正方形之外的长方形共有 SumR = SumA-SumS
