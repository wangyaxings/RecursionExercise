private static int AddAllDigits(int num)
{
    int sum = 0;
    if (num<10)
    {
        return num;
    }
    else
    {
        sum = sum + num%10 + AddAllDigits(num/10);
    }
    return sum;
}
按计算机实际执行的递归顺序思考的话，想几步思维就乱，
遇到递归的问题，你可以描述出来时，如此：
它的描述：将该整数除以10的商持续除以10，直到商值小于10，将每次的余数相加（当然包括最后的个位），
实际代码编写的过程中，
首先写出终止条件和终止条件之外的其他条件else，按照单次执行逻辑写出部分代码，通过以该递归函数（AddDigits(num/10)）为代理，代替其它所有次相同操作逻辑
从而完成递归。
